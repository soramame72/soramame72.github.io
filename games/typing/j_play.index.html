日本語タイピングゲーム

流れてくる単語を正確に入力しよう！

遊び方

  • スタートボタンを押してゲーム開始
  • 流れてくる単語をローマ字で入力
  • 青色にハイライトされた文字が次に入力する文字です
  • ESCキーでゲーム終了

対応している入力方法

? しゃ → sha / sya

? ちゃ → cha / tya

? じゃ → ja / jya / zya

? ふ → fu / hu

? つ → tu / tsu

? っ → xtu / xtsu / ltu / ltsu / 次の文字の繰り返し

? ん → n / nn (母音の前など自動判定)

? を → wo / o
